# 4. Report Creation

## Interactive Dashboard

The file `email_crm_dashboard.html` is a fully self-contained interactive dashboard. Open in any web browser.

## Dashboard Sections

### Section 1 — Email Funnel
- Full funnel visualization: Sent → Delivered → Opened → Clicked → Converted
- Funnel drop-off rates at each stage

### Section 2 — Campaign Type Comparison
- Open rate, CTR, conversion rate by campaign type
- Revenue per email by type
- Volume distribution

### Section 3 — Segmentation Performance
- Engagement metrics by customer segment
- Revenue concentration by segment

### Section 4 — Timing & Device
- Open rate by hour and day of week
- Device type distribution
- Best performing send times

## Key Metrics

| KPI | Description |
|-----|-------------|
| Delivery Rate | % of sent emails successfully delivered |
| Open Rate | % of delivered emails opened |
| CTR | % of opened emails with clicks |
| Conversion Rate | % of clicks leading to purchase |
| Revenue per Email | Total revenue / emails sent |
| Unsubscribe Rate | % opting out per campaign |
