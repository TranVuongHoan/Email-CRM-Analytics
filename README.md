# 📧 Email & CRM Campaign Analytics

This repository contains a comprehensive analysis of email marketing and CRM campaign performance data extracted from `02_Email_CRM_Campaign.xlsx`. The analysis covers email engagement metrics, customer segmentation, and campaign ROI across the full customer lifecycle.

---

## 📁 Repository Structure

| Folder | Contents |
|--------|----------|
| `1. Data Preparation/` | Dataset overview, field descriptions |
| `2. Data Cleaning/` | Data normalization and quality fixes |
| `3. Data Analysis/` | 20 SQL queries + 7 Python EDA charts |
| `4. Report Creation/` | Interactive HTML dashboard |

---

## 🛠️ Tools Used
- **Python** — pandas, matplotlib
- **SQL** — DuckDB/PostgreSQL compatible
- **HTML/JS/Chart.js** — Interactive dashboard
